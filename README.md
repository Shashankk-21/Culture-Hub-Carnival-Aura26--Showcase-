The official Cultural Club Showcase portal for the Aura '26 Carnival of Cultures. This platform connects students with cultural clubs, events, and registrations.

##  Features
* **Dynamic Club Profiles:** Detailed pages for Music, Dance, Drama, Art, Literature, and Photography clubs.
* **Event Registration:** Integrated enquiry system for club memberships and event participation.
* **Responsive Design:** Fully optimized mobile navigation with hamburger menu support.
* **Carnival Theme:** Custom Tailwind CSS implementation using the official Aura '26 color palette (Teal, Gold, Deep Purple).
* **Data-Driven UI:** Component-based architecture rendering dynamic content from a centralized data store.

##  Tech Stack
* **Frontend:** React.js, Tailwind CSS, Lucide-React
* **Routing:** React Router DOM (Client-side routing)
* **State Management:** Local Storage (for persisting user preferences)

##  How to Run
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:5173` (or the port shown in terminal).
